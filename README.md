# Sandbox

------------------
####  3 Node CentOs7 Sandbox
1 Master - 2 nodes

------------------

Sandbox for learning devOps related technology. Starting with Ansible


## Configured
User admin priveledges & Public/Private Keys are created and linked from master to each node.

```
scripts/
  -- add_keys.sh
  -- add_user.sh
  -- configure_ssh.sh
```


##  Installation

```
vagrant up
```
